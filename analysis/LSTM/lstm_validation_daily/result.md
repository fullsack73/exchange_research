### 분석 목적
일간 데이터 기준으로 전체기간과 이상기간을 분리해서, 금리차 중심 모델과 MMF 포함 모델의 성능을 비교해 기존 가설을 재검증하기

### 분석 방법
LSTM 시계열 예측 모델을 동일 조건으로 두 번 실행
- 전체기간: 2010년부터 2025년
- 이상기간: 2024년 11월부터 2025년 12월

각 기간에서 Model A(Spread only)와 Model B(Spread + MMF)를 RMSE, MAE로 비교

### 분석 결과
전체기간에서는 Model A가 더 우수하게 나옴. 즉 정상구간까지 포함한 장기 데이터에서는 금리차 정보만으로도 평균적인 추세를 따라가는 힘이 강함.

반면 이상기간만 떼어보면 Model B가 RMSE와 MAE 모두 더 좋게 나옴. 즉 우리가 분석 대상으로 잡은 비정상 구간에서는 MMF 정보를 추가했을 때 환율 급등 구간을 더 정확하게 포착함.

결론적으로 비교 기준을 월간 vs 일간이 아니라 정상구간 포함 전체기간 vs 이상기간으로 바꾸면, 이상기간에서 MMF 가설이 더 설득력 있게 확인됨.
